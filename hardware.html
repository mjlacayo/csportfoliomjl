CPU

The central processing unit, or CPU, is like the brain of the computer. It performs calculations when the user performs basic operations on the computer. Depending on the brand and/or specifications of the CPU the computer can perform actions slower or faster.

RAM

Random access memory, or RAM, is where memory that the CPU is using in the moment is stored. Once the computer is shut off the RAM is wiped clean. One way to think of the ram is as the short term memory of the computer.

Motherboard

The motherboard is where all the components of a computer meet and work together to make the computer work properly. While the motherboard does not affect the performance of a computer, it can limit the potential power and capability of a machine.

Hard Drive

The hard drive is similar to RAM in the sense that it stores data, however it is more like long term memory and not short term. People most comonly use hard drives for mass storage of documents, pictures/images, and applications. Hard drives come in a wide range of data capacity from 100 gigabytes to 4 terabytes.

Peripherals (Mouse, Keyboard, and Monitor)

For most computers to be used they need to be accompanied by things called peripherals. The three major peripherals are the mouse, keyboard, and monitor. The monitor is what displays the computer screen, the mouse is how you move your cursor across the screen, and the keyboard is what allows you to input text and commands into the computer.
