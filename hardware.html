    • CPU

    The central processing unit, or CPU, is like the brain of the computer. It performs calculations when the user performs basic operations on the computer. Depending on the brand and/or specifications of the CPU the computer can perform actions slower or faster.

    RAM

    Random access memory, or RAM, is where memory that the CPU is using in the moment is stored. Once the computer is shut off the RAM is wiped clean. One way to think of the ram is to think of it as the short term memory of the computer.

    Motherboard

    The motherboard is like the frame of a house before insulation, flooring, walling, electrical, and plumbing have been put in. While the motherboard does not affect the performance of a computer, it can limit the potential power and capability of a machine.
